手机运行c程序

2020-01-22 如何用Android手机Linux模拟器编写和运行c程序

安装clang编译器
安装vim编辑器或者使用Linux自带的nano（字符终端文本编辑器）
最后命令./文件名（不带后缀）./a.out运行该可执行程序
注意：当执行vi命令创建文件时要带后缀。
gcc 文件名带后缀 -o 文件名不带后缀 与 clang 文件名带后缀 -o 文件名不带后缀 都可以
附带：https://ipcmen.com/nano（nano指令大全）
https://www.runoob.com/linux/linux-vim.html（vim指令大全）
